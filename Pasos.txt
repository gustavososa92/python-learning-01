* Tener instalado Docker

* Crear una carpeta por cada container
    * Crear el archivo "Dockerfile" (sin extension)
    * Crear el archivo "requirements.txt"
    * Completar el Dockerfile con las imagenes que necesito
    * En requirements poner las dependencias del proyecto

* Crear el archivo "docker-compose.yml" y configurar la version y los servicios
    * A cada servicio darle un nombre, el path para buildear el Dockerfile y el puerto a exponer

* Hacer docker-compose build

* Hacer docker-compose up

* ports y network_mode: host no son compatibles (ya no...)


* Crear un contenerdor para mongo
    Log in al container: 
            docker exec -it mongodb bash
            docker exec -it web_api_rest bash

    Autenticarse en mongo: 
            mongosh -u admin -p password --authenticationDatabase firstdb
            
    Mongo URL 
            mongodb://admin:password@127.0.0.1:27017/firstdb
            mongodb://admin:password@127.0.0.1:27017

* Starting from Mongodb version 6.0 mongo was replaced by mongosh
